# risu/bot — Bot & Character Authoring

Composition guidance for writing and improving `.charx` bots and character cards.

## What lives here

- `skills/` — bot composition skills
- `docs/` — critique frameworks and bot-specific reference material
- `AGENTS.md` — thin routing surface for AI assistants
- local bot project folders — ignored work products and references

## Scope boundary

| Here                                                                                                             | Elsewhere                                                                                        |
| ---------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------ |
| Character description structure, self-introduction sheet methods, lorebook-driven bot architecture, bot critique | Shared syntax like CBS, lorebook schema, Lua APIs, regex, RisuAI HTML/CSS, and Arca/WYSIWYG HTML |

## Start here

- `AGENTS.md`
- `skills/README.md`
- `docs/`
- `../common/skills/README.md`
